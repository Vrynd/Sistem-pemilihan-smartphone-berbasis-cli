# Sistem Pemilihan Smartphone

Aplikasi Python untuk membantu pengguna memilih smartphone yang sesuai dengan kebutuhan mereka.

## Fitur
- Analisis kebutuhan pengguna
- Rekomendasi smartphone berdasarkan kriteria
- Perbandingan spesifikasi
- Proses pemilihan yang interaktif

## Instalasi

1. Clone atau download repository
2. Buat virtual environment:
```bash
python -m venv venv
```

3. Aktifkan virtual environment:
   - Windows: `venv\Scripts\activate`
   - Linux/Mac: `source venv/bin/activate`

4. Install dependencies:
```bash
pip install -r requirements.txt
```

## Cara Menjalankan

```bash
python main.py
```

## Struktur Folder
```
├── main.py              # File utama aplikasi
├── requirements.txt     # Daftar dependencies
├── README.md           # Dokumentasi ini
└── venv/              # Virtual environment (jangan commit)
```

## Teknologi
- Python 3.x
- (Dependencies akan ditambahkan sesuai kebutuhan)
